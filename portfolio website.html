Personal Portfolio Website

A fully responsive portfolio website built from scratch using HTML, CSS, and JavaScript to showcase my work.

Project Overview

This custom-built portfolio website was designed to showcase my professional work, skills, and personality in a clean, modern interface. The goal was to create a fully responsive website that provides an excellent user experience across all devices while highlighting my capabilities as both a designer and developer.

Key features include smooth animations, interactive elements, a dark/light mode toggle, project filtering system, and a contact form. The design follows modern web standards with attention to accessibility and performance optimization. The website was built mobile-first to ensure perfect responsiveness on all screen sizes.
